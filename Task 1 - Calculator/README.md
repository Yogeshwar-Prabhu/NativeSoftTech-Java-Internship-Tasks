# Task 1 – Simple Calculator

This is a web-based calculator developed using HTML, Tailwind CSS, and JavaScript.

## 🛠️ Features:
- Perform basic arithmetic operations (+, −, ×, ÷)
- Responsive UI designed for desktop & mobile
- Dark Mode toggle

## 📦 Technologies Used:
- HTML5
- Tailwind CSS
- Vanilla JavaScript
